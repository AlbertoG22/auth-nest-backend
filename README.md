# Backend en Nest

## Pasos:
1. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
2. Llenar las variables de entorno con tus valores.
3. Ejecutar los servicios definidos en el archivo ```docker-compose.yml```
```
docker compose up -d
```

4. Para development ejecutar:
```
npm run start:dev
```
